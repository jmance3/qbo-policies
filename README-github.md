# QuickBooks Integration Policies

This repository hosts the Privacy Policy and End User License Agreement (EULA) for South Fork Design Build's internal QuickBooks integration.

## Files

- **privacy.html** - Privacy Policy for QuickBooks data handling
- **eula.html** - End User License Agreement for the integration

## Purpose

These documents are required by Intuit to enable production access for the QuickBooks MCP integration. They are hosted publicly per Intuit's requirements but govern internal use only.

## Usage

These policies apply to:
- South Fork Design Build's QuickBooks MCP Server
- Internal invoice generation and financial workflows
- Authorized team members only
